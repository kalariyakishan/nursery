# 🚀 Smart Pantry / Invoice System - Local Setup Guide

## 📌 Requirements

Before starting, make sure:

* XAMPP is installed in `C:\xampp`
* Apache & MySQL are not already running

---

## ▶️ How to Start the Project (Very Easy)

### ✅ Option 1 (Recommended)

1. Go to `devdoc` folder
2. Double click on:
   👉 `run_project.vbs`

✔ This will:

* Start XAMPP (Apache + MySQL)
* Open the system in browser automatically

---

### ✅ Option 2 (Manual)

1. Double click:
   👉 `start_project.bat`

---

## 🌐 Project URL

Once started, system will open automatically.

If not, open manually in browser:

👉 http://localhost/your-project/public

---

## ❗ Important Notes

* Do NOT move project folder outside `htdocs`
* Do NOT rename project folder after setup
* Always run using provided files (no technical steps needed)

---

## 🛠 Troubleshooting

### If project does not open:

* Check if XAMPP is installed in `C:\xampp`
* Start Apache & MySQL manually from XAMPP Control Panel

### If port issue:

* Make sure no other software is using port 80 or 3306
